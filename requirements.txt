# ==============================================================
# HOW TO RUN
# ==============================================================
#
# 1. Install dependencies:
#    pip install -r requirements.txt
#
# 2. Set your Gemini API key:
#    export GOOGLE_API_KEY="your-key-here"
#
# 3. Place all files in this structure:
#
#    project/
#    ├── app/
#    │   ├── __init__.py      ← empty file
#    │   ├── main.py
#    │   ├── models.py
#    │   ├── storage.py
#    │   ├── job_store.py
#    │   ├── ingest.py
#    │   └── retrieval.py
#    ├── storage/             ← auto-created on startup
#    ├── requirements.txt
#    └── run.py
#
# 4. Run the server:
#    uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
#
# 5. Open Swagger UI to test all endpoints:
#    http://localhost:8000/docs
#
# 6. Test flow:
#    a. POST /ingest      → upload PDF → get {job_id, doc_id}
#    b. GET  /status/{job_id} → poll until status="done"
#    c. POST /query       → {question, doc_id} → get {answer, sources}
#    d. Open image URL    → verify table/chart image renders
#
# ==============================================================
# CONCERNS ADDRESSED
# ==============================================================
#
# Concern 1 — Duplicate PDF detection     → MD5 hash check in /ingest
# Concern 2 — Query result caching        → _query_cache dict in retrieval.py
# Concern 3 — Large file upload limit     → 50MB cap + streaming in main.py
# Concern 4 — Pipeline crash cleanup      → try/except + full rollback in ingest.py
# Concern 5 — Job state persistence       → JobModel in SQLite via job_store.py
# Concern 6 — Async event loop safety     → ThreadPoolExecutor in main.py
# Concern 7 — Image path portability      → relative paths in DB, URLs at serve time
# Concern 8 — CORS                        → CORSMiddleware in main.py
